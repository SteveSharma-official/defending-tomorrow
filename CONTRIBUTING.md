# Contributing to Defending Tomorrow

Thank you for your interest in improving this resource!

## Types of Contributions Welcome

- 🐛 **Bug fixes** for code examples
- 📝 **Clarifications** to comments or documentation
- 🧪 **Additional test cases** for existing code
- 🌐 **Translations** of checklists (community)
- 🔒 **Security vulnerability disclosures** (private channel)

## Pull Request Process

1. Fork the repository
2. Create a feature branch (`git checkout -b improve-chapter3`)
3. Make your changes
4. Run tests if applicable
5. Submit a Pull Request against `main`

## Code Style

- **Python:** Follow PEP 8, use type hints
- **Comments:** Include docstrings for functions
- **Filename:** Lowercase with underscores

## Reporting Security Issues

**Do not open public issues for security vulnerabilities.**

Contact: security@stevesharma.com (PGP key available upon request)
